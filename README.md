# Hotel-management-system
This project is about a booking system in a hotel.
It contains multiple pages where the user can search for a room and book it with the description,budget and date that he wants.
It also contains an additional oage for the admin where he can put the password and username to access it,there he can add a new manager,book a room or edit a room already booked and to see the different room categories.
This project was made using php,css,and sql for the database.
